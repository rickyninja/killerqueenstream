# ffmpeg

As of Debian stretch, which raspbian is based on, the real ffmpeg is available via package instead of avconv.
The package also includes support for using raspberry pi's GPU hardware acceleration for h264 encoding.

`sudo apt install ffmpeg`


You can verify the hardware acceleration is available to ffmpeg via `--enable-omx-rpi` and `--enable-mmal`.
